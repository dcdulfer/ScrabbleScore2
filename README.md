# Scrabble Score Tracker

A mobile-first web application for tracking Scrabble game scores with advanced features including word calculator, statistics tracking, and achievements.

## Features

- **Score Tracking**: Real-time score tracking for multiple players
- **Quick Score Buttons**: Pre-set buttons for common scores (+10, +20, +50, +75, +100)
- **Custom Scores**: Add any custom score value
- **Undo Function**: Revert the last score addition
- **Word Calculator**: Calculate word scores using Scrabble letter values
- **Game Statistics**: Track games played, total time, highest scores
- **Achievements**: Unlock achievements as you play
- **Game History**: View past games and results
- **Local Storage**: All data persists in your browser

## How to Use

1. Click "New Game" to set up players and game settings
2. Add players with custom names and colors
3. Set a target score (100-500 points)
4. Choose a game mode (Standard, Timed Challenge, Tournament)
5. Track scores during gameplay
6. Use the word calculator to compute word values
7. View statistics and achievements in the Stats tab

## Deployment

This app is hosted on GitHub Pages and requires no server-side setup.

## Technologies

- React 18
- Tailwind CSS
- Lucide React Icons
- Local Storage for persistence
